# Screenshots Directory

Direktori ini berisi bukti screenshot validasi untuk studi kasus SE-1 Framework.

## Required Files

- `prediction-may-9-2026.png` - Postingan prediksi 9 Mei 2026
- `validation-may-20-2026.png` - Berita perjanjian damai 20 Mei 2026
- `validation-june-2-2026.png` - Berita negosiasi ditunda 2-3 Juni 2026
- `analysis-may-5-2026.png` - Analisis fundamental 5 Mei 2026 (optional)

## How to Add Screenshots

1. Click "Add file" → "Upload files"
2. Drag and drop your screenshot files
3. Commit changes

## File Naming Convention

Gunakan format: `[type]-[date]-[description].png`

Contoh:
- `prediction-2026-05-09-iran-conflict.png`
- `validation-2026-05-20-peace-agreement.png`
- `validation-2026-06-02-negotiation-delay.png`
